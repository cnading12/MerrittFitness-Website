# Project Memory — Merritt Wellness

## Business name
The business is **Merritt Wellness** — NOT "Merritt Fitness". The git repo is
named `MerrittFitness-Website` for historical reasons, but all customer-facing
copy, documents, and branding must say **Merritt Wellness**.

## Brand palette
Defined in `app/globals.css`. The brand is warm taupe/cream — **not green**.

| Token | Hex | Use |
|-------|-----|-----|
| `--brand-primary` | `#735e59` | Warm taupe — buttons, links, headings |
| `--brand-primary-dark` | `#5a4a46` | Hover / darker accents |
| `--brand-primary-light` | `#8a7571` | Secondary accents |
| `--brand-cream` | `#f2eee9` | Main background |
| `--brand-cream-dark` | `#e8e2db` | Cards / sections |
| `--brand-warm-white` | `#faf8f5` | Body background |
| `--brand-text` | `#4a3f3c` | Primary body text |
| `--brand-text-light` | `#6b5f5b` | Secondary text |
| `--brand-accent` | `#a08b84` | Labels, muted text |
